# Лабораторные по основам информационной безопасности

Репозиторий для лабораторных работ по курсу «Основы информационной безопасности».

## Содержание

- Описание и цель лабораторных работ
- Инструкция по запуску и тестированию
- Структура проекта

## Структура проекта (черновик)

- `labs/` — лабораторные работы
- `docs/` — вспомогательные материалы и конспекты
- `scripts/` — служебные скрипты для сборки/запуска (при необходимости)

## Начало работы

1. Установите необходимые зависимости (если есть файлы `requirements.txt`, `package.json` и т.п.).
2. Ознакомьтесь со структурой директорий.
3. Выполняйте лабораторные работы в соответствующих подпапках.
